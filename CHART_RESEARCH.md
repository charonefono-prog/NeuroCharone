# Padrões de Gráficos em Publicações Científicas Clínicas

## Observações das imagens de referência:

### Gráfico de Barras Pré/Pós (Nature, ResearchGate):
- Barras lado a lado com cores distintas (azul=pré, verde/laranja=pós)
- Eixo Y com rótulo claro (ex: "Score", "Pontuação")
- Eixo X com rótulo das avaliações (ex: "Pré", "Pós", "Follow-up")
- Valores numéricos exibidos ACIMA de cada barra
- Linhas de grade horizontais sutis
- Legenda clara no topo ou inferior
- Fundo branco limpo
- Asteriscos para significância estatística (*, **, ***)
- Barras de erro (error bars) quando aplicável

### Gráfico de Tendência/Evolução (PHQ-9, Nature):
- Linha contínua com marcadores circulares nos pontos de dados
- Eixo Y com escala do score (0 a max)
- Eixo X com datas ou timepoints
- Faixas horizontais coloridas indicando severidade (ex: Normal, Leve, Moderado, Severo)
- Título descritivo no topo
- Fonte serif ou sans-serif profissional
- Cores sóbrias (preto, azul escuro, cinza)

### Terminologia Científica:
- "Baseline" ou "Pré-tratamento" (não "Primeiro score")
- "Follow-up" ou "Pós-tratamento" (não "Último score")
- "Variação" ou "Diferença" (não genérico)
- "Redução clinicamente significativa" (não "Melhora")
- "Sem alteração significativa" (não "Estabilidade" - que pode confundir com estar bem)
- "Manutenção do quadro" (quando score não muda)

### Cores Padrão Científico:
- Azul escuro (#2563EB ou #1E40AF) para baseline/pré
- Verde (#16A34A) para melhora
- Vermelho (#DC2626) para piora
- Cinza (#6B7280) para neutro/sem alteração
- Fundo branco (#FFFFFF)
- Linhas de grade cinza claro (#E5E7EB)
