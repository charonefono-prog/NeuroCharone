## 📋 INSTRUÇÕES PARA USAR O TEMPLATE

### Passo 1: Preparar a Lista de Clientes
1. Exporte a lista de assinantes anuais que NÃO compraram o capacete
2. Separe os e-mails e nomes dos clientes

### Passo 2: Personalizar a Mensagem
1. Copie a versão curta ou completa acima
2. Substitua `[INSERIR EMAIL DO CLIENTE]` pelo e-mail real
3. Substitua `[NOME DO CLIENTE]` pelo nome real (se usar versão completa)

### Passo 3: Enviar via WhatsApp
**Opção A - Manual:**
1. Abra WhatsApp
2. Procure pelo contato do cliente
3. Cole a mensagem personalizada
4. Envie

**Opção B - Automático (usando WhatsApp Business API):**
1. Use uma ferramenta como Twilio, MessageBird ou similar
2. Configure para enviar a mensagem personalizada para cada cliente
3. Agende o envio para um horário específico

### Passo 4: Acompanhar Respostas
1. Monitore as respostas dos clientes
2. Responda dúvidas e problemas
3. Registre quem baixou e quem precisa de suporte
