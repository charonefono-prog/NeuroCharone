## Fórmula de Melhora

### Para escalas INVERSAS:
- Se score diminuiu: `melhora = ((antes - depois) / antes) * 100` → positivo = melhorou
- Se score aumentou: `piora = ((depois - antes) / antes) * 100` → negativo = piorou

### Para escalas DIRETAS:
- Se score aumentou: `melhora = ((depois - antes) / antes) * 100` → positivo = melhorou
- Se score diminuiu: `piora = ((antes - depois) / antes) * 100` → negativo = piorou

### Caso especial: antes = 0
- Se antes = 0 e depois = 0: variação = 0%
- Se antes = 0 e depois ≠ 0: variação = 100% (melhora ou piora dependendo da direção)
