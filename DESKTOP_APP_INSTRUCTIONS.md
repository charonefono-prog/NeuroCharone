# NeuroLaserMap - Desktop App para Windows

## Como Usar

### Opção 1: Executar Direto (Mais Fácil)
1. Descompacte a pasta `desktop/`
2. Abra o Prompt de Comando (CMD) na pasta
3. Digite: `npm start`
4. O app vai abrir automaticamente!

### Opção 2: Gerar Instalador (.exe)
1. Descompacte a pasta `desktop/`
2. Abra o Prompt de Comando (CMD) na pasta
3. Digite: `npm run build`
4. Aguarde alguns minutos
5. O instalador `.exe` vai aparecer em `desktop/dist/`
6. Clique no `.exe` para instalar

## Requisitos
- **Node.js** instalado (https://nodejs.org/)
- **Python** instalado (para o servidor web rodar)
- **Servidor rodando** em http://localhost:3000/
