## 💻 Versão Desktop (Windows)

### Opção 1: Executar Direto (Mais Fácil)
1. Abra a pasta `desktop/`
2. Abra Prompt de Comando (CMD) aqui
3. Digite: `npm start`
4. O app abre automaticamente!

### Opção 2: Gerar Instalador (.exe)
1. Abra a pasta `desktop/`
2. **Clique 2x em `BUILD_WINDOWS.bat`**
3. Aguarde alguns minutos
4. O instalador aparece em `desktop/dist/`
5. Clique no `.exe` para instalar

## 👥 Controle de Usuários (Painel Admin)

### Como Funciona:
1. **Você faz login como ADMIN**
2. **Clica na aba "Admin"**
3. **Vê lista de usuários com status:**
   - 🟡 Pendente (novo usuário)
   - ✅ Aprovado (ativo)
   - ❌ Bloqueado (sem acesso)

### Ações:
- ✅ **Aprovar** - Libera acesso
- 🚫 **Bloquear** - Tira acesso
- 🗑️ **Deletar** - Remove do sistema
